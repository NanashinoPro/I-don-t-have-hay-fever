# 組込コマンド

## 概要

すべてのプレーヤーが使用することができるコマンド。
これらを駆使することで、多種多様な戦い方を実践することができる。

## コマンド一覧と、それぞれの役割
| コマンド名                                   | 役割                              | XWorldでの能力 |
|---------------------------------------------|-----------------------------------|------|
| `job_spec [&]`                              | ジョブをバックグラウンドで実行      |      |
| `(( expression ))`                          | 数式の評価                        |      |
| `. filename [arguments]`                    | ファイルを現在のシェル環境で実行    |      |
| `:`                                         | 空のコマンド                      |      |
| `[ arg... ]`                                | 条件式                            |      |
| `[[ expression ]]`                          | 拡張条件式                        |      |
| `alias [-p] [name[=value] ... ]`            | エイリアスの設定・表示            |      |

| `bind [-lpsvPSVX]...`                       | キーバインディングの設定・表示      |      |
| `break [n]`                                 | ループを終了                      |      |

| `case WORD in [PATTERN...] ... esac`        | パターンマッチング                |      |
